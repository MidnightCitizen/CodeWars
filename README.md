## General info
Continuous training with problems solving from https://www.codewars.com/dashboard

## Technologies
* Language used : Javascript
* Test Driven Development : jest

## Setup

Install it locally using npm:

```
$ npm i
```

Run all the tests
```
$ npm run test
```

Run only one test 
```
$ npm test -- nametest.test.js
```
